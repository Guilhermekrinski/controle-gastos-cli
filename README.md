# Controle de Gastos CLI

## 📌 Problema
Muitas pessoas têm dificuldade em controlar seus gastos diários, o que pode levar a desorganização financeira e até endividamento.

## 💡 Solução
Esta aplicação permite registrar, listar e calcular gastos de forma simples utilizando o terminal.

## 👥 Público-alvo
Pessoas que desejam organizar suas finanças pessoais de forma prática.

## ⚙️ Funcionalidades
- Adicionar gasto
- Listar gastos
- Ver total de gastos

## 🚀 Tecnologias
- Node.js
- Jest (testes)
- ESLint (qualidade de código)
